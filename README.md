# Kurogane Security Model

Kurogane Security Model documents the public security, transparency, and review model for the public technical ecosystem around Kurogane Hub.

Kurogane Hub is MKDL's private commercial OT cybersecurity and industrial monitoring platform. The commercial production core remains private, while selected auditors, partners, investors, and security researchers may request controlled technical review access.

## Scope

- High-level threat model.
- Trust boundaries between hub, satellite, customer, operator, OT network, and IT network.
- Responsible disclosure process.
- SBOM strategy.
- Credential handling principles.
- Network assumptions and hardening principles.
- Security review checklist.
- Data privacy model.

## Boundaries

This repository does not publish production code, sensitive detection logic, customer deployment logic, internal endpoints, credentials, IP addresses, real plant data, or details that would help attack a real installation.
